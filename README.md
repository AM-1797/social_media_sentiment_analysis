# social_media_sentiment_analysis

**Social Media Sentiment Analysis on Brand Tweets**

**Overview**
This project analyzes public sentiment about brands using Twitter data. It collects tweets, cleans and processes text, performs sentiment analysis (VADER + TextBlob), and visualizes results with charts and word clouds.

**Tools Used**
- **Python**
- **Tweepy** for Twitter API data extraction
- **NLTK & TextBlob** for NLP & sentiment analysis
- **Matplotlib / Seaborn** for data visualization
- **Streamlit** for interactive dashboard

## Features
- Extract tweets by brand keyword
- Clean text data (remove hashtags, mentions, URLs)
- Sentiment classification (Positive / Negative / Neutral)
- Generate WordClouds and visual charts
- Optional: Live dashboard with Streamlit
